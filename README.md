# RuneBrowser
RuneBrowser is an addon hand-tailored for GrimfallWoW. It allows the user to browse between the nigh endless amount of Runes from the Rune system that the project offers.

**What RuneBrowser does:**
RB lists all currently existing (and relevant) runes found in the gamefiles.

**What RuneBrowser doesn't do:**
RB has no "show known/unknown runes" option, as it is merely a browser. Think of something like AtlasLoot for example.

When placing into AddOns folder, remove the "-main" part from the Folder's name.
